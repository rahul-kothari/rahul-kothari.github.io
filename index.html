Image Panorama Stitching

Designed a Computer Vision algorithm that stitches ANY number of images(even at different rotations or scale) together to form a panorama.

The algorithm uses

  • image keypoint detection using SIFT,
  • then matches the keypoints and
  • stitches a pair of images using RANSAC and Homography matrices.
  • This can be extended to any number of images.
